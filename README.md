# Movies Data Analysis Project

## Objective
Analyze movie data to understand factors affecting profitability and ROI.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Power BI

## Key Insights
- Higher budget movies generally generate more revenue but do not guarantee success
- Animation genre generates highest profit
- ROI is higher in low-budget genres like comedy
- Ratings do not strongly affect financial returns

## Dashboard
Power BI dashboard created to visualize:
- Profit by genre
- Budget vs revenue
- ROI distribution
- Year-wise trends

## Conclusion
This project highlights how data analysis can help in making better business decisions in the movie industry.
